## Overview

Find Person API provides the person's details, including demographics, all current and historical addresses, phone numbers, and relatives and associates to this person.

## Endpoint

<RequestExample>

```bash
GET https://api.trestleiq.com/3.1/person?name={name}&address.city={city}&address.state_code={state}
```

</RequestExample>

## Parameters

<ParamField path="name" body="query" type="string" required>
  The name of the person to search. **Example:** `John Doe`
</ParamField>

<ParamField path="address.street_line_1" body="query" type="string">
  The first line of the street part in the structured address.
</ParamField>

<ParamField path="address.street_line_2" body="query" type="string">
  The second line of the street part in the structured address.
</ParamField>

<ParamField path="address.city" body="query" type="string">
  The name of the city in the structured address. **Example:** `Seattle`
</ParamField>

<ParamField path="address.postal_code" body="query" type="string">
  The postal code of the structured address. **Example:** `98101`
</ParamField>

<ParamField path="address.state_code" body="query" type="string">
  The state code of the structured address. **Example:** `WA`
</ParamField>

<ParamField path="address.country_code" body="query" type="string">
  The ISO-3166 alpha-2 country code of the address. **Example:** `US`
</ParamField>

## Headers

<ParamField path="x-api-key" body="header" type="string" required>
  Your API key for authentication.
</ParamField>

## Code Examples

<CodeGroup>

```bash cURL
curl --request GET \
```

```javascript JavaScript
const response = await fetch(
```

</CodeGroup>

## Response

<ResponseField name="count_person" type="integer">
  The number of people returned.
</ResponseField>

<ResponseField name="person" type="array">
  An array of people associated with the address provided. Each person object includes: - Demographics - All current and historical addresses - Phone numbers - Relatives and associates
</ResponseField>

<ResponseField name="error" type="object">
  Error information if the request failed.
</ResponseField>

<ResponseField name="warnings" type="array">
  Warnings returned as part of the response. Possible values include: `Missing
      Input`, `Invalid Input Address`, `Invalid house/building number`, `Missing
      unit/apt/suite number`, or `Invalid unit/apt/suite number`.
</ResponseField>
