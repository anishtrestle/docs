## What is Trestle?

Trestle provides identity data for businesses. This identity data is key for building and maintaining great customer relationships.

## Base URL

All API requests should be made to:

```
https://api.trestleiq.com
```

## Available APIs

Trestle offers a comprehensive suite of identity data APIs:

### Reverse Phone API

Find the current resident or business associated with a phone number and their information, including demographic data, address, associated people and more.

### Caller Identification API

Find the single owner and location of the person or business calling in. Perfect for real-time caller identification.

### Smart CNAM API

Validates a phone number and provides person or business name associated to the phone. Lightweight and fast.

### Phone Validation API

Lightweight phone check to identify line type, carrier, validity, and more. Essential for data quality and fraud prevention.

### Reverse Address API

Find the current resident or business associated with an address and their information, including demographic data, phones, associated people and more.

### Find Person API

Find a person's details, including demographics, all current and historical addresses, and phone numbers.

### Real Contact API

Validates whether a lead is real, confirms the person is who he/she says they are, and grades the contactability of the lead.

## Getting Started

1. **Get an API Key**: [Sign up for a Developer Portal account](https://portal.trestleiq.com/signup)
2. **Review Authentication**: See our [Authentication guide](/guides/authentication)
3. **Try an API**: Start with the [Reverse Phone API](/api-reference/reverse-phone-api) or [Phone Validation API](/api-reference/phone-validation-api)
4. **Check Status**: Monitor API status at [status.trestleiq.com](https://status.trestleiq.com)
